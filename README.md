# Eclipse Club System

Веб-приложение для учёта посещаемости и активности участников танцевального клуба Eclipse.

## Зачем

Раньше учёт участников клуба вёлся в заметках и по памяти. Приложение собирает это в одном месте:
кто состоит в клубе, сколько репетиций посетил, сколько было выступлений, и насколько активен участник.

## Что умеет

- вход администратора
- добавление и редактирование участников
- автоматический расчёт посещаемости
- автоматический расчёт балла активности
- отчёт по каждому участнику
- данные сохраняются в браузере

## Формулы

- Посещаемость = посещённые репетиции / всего репетиций × 100%
- Балл активности = выступления × 5 + посещённые репетиции

## Как запустить

Открыть ссылку в браузере на телефоне или компьютере. Установка не нужна.

Логин: `admin` · Пароль: `1234`

## Технически

Один HTML-файл: HTML, CSS и JavaScript без библиотек и без сервера.
Данные хранятся локально в браузере (localStorage) и никуда не отправляются.

Первая версия была написана на C# WinForms как школьный проект по информатике;
эта веб-версия сделана, чтобы приложением можно было пользоваться с любого устройства.

## Ограничения

- данные хранятся только в том браузере, где их ввели
- один администратор, простой пароль (учебный проект)
- нет экспорта в PDF
