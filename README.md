# Eclipse Club System

Веб-приложение для управления танцевальным клубом Eclipse: посещаемость, выступления и активность участников.

**Открыть:** https://admit-socrates.github.io/takhmina-sultanbekova-eclipse-club-system/ (пароль `1234`)

## Задача

Раньше учёт участников клуба вёлся в заметках и по памяти. Было непонятно, кто ходит стабильно,
а кто пропал, и кого можно ставить в номер. Приложение отвечает на эти вопросы автоматически.

## Как это работает

Главная идея: не вводить итоговые числа вручную, а отмечать реальные события.

1. Добавляем участников клуба
2. После каждой репетиции отмечаем галочками, кто пришёл
3. Каждое выступление сохраняем вместе с составом

Посещаемость и балл активности считаются сами из этих записей.

## Возможности

- обзор клуба: количество участников, средняя посещаемость, репетиции, выступления
- список «Нужно внимание»: участники с посещаемостью ниже 60%
- рейтинг самых активных
- карточка участника с историей выступлений
- поиск и сортировка (по имени, посещаемости, баллу)
- выгрузка в CSV
- работает с телефона

## Формулы

- Посещаемость = посещённые репетиции / всего репетиций × 100%
- Балл активности = выступления × 5 + посещённые репетиции

## Технически

Один HTML-файл: HTML, CSS и JavaScript. Без библиотек, без сервера, без сборки.
Данные хранятся в браузере (localStorage) и никуда не отправляются.

Первая версия была написана на C# WinForms как школьный проект по информатике.
Веб-версия сделана, чтобы приложением можно было пользоваться с любого устройства.

## Ограничения

- данные хранятся только в том браузере, где их ввели — общей базы нет
- один руководитель, простой пароль (учебный проект, не система безопасности)
- нет экспорта в PDF

Если развивать дальше: общая база данных, чтобы участники видели свою статистику сами.
